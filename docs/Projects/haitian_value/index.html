    • 制定与修订企业战略，选择合理的战略经营方向

      • 充分调研市场信息，进入新兴细分行业（例如新能源汽车）
      • 发掘新的海外市场，持续扩大海外市场份额
      • 分析企业经营瓶颈，适当进行战略性产业并购（横向一体化/纵向一体化）
      • 结合企业长期发展需求，进行新的股权融资（其他产业的上市）或发行境内外长期债券

    • 将企业战略目标分解形成经营计划及预算目标

      • 将长期发展战略分解形成可执行的经营计划

      • 通过预算管理，将经营计划转化为可对比分析的财务数字

        • 财务

          • 年度预算编制及时率

    • 完善经营分析体系建设，及时发现各项经营管理问题

      • 按照经营分析需求形成内部管理报告，为上层决策提供可靠的分析数据
      • 按照相关财务章程与外部合规要求，形成全面准确的外部财务报告

    • 建立绩效考核及战略回顾机制，形成战略调整依据

      • 将战略目标转化分解为考核指标，建立闭环的绩效考核机制
      • 结合经营分析报告与绩效考核结果，及时调整相关经营目标与绩效指标

  • 项目管理体系建设

    • 建立及编写项目管理体系整体制度

      • 明确项目管理机制适用范围及应用标准

    • 项目立项评审与结项复盘机制

      • 建立项目分级评审委员会，审核不同项目的立项申请
      • 建立项目结果分析制度，对出现异常结果的项目进行复盘分析并撰写分析报告

    • 项目计划管理机制

      • 完善项目的全计划制定，明确整体周期及各里程碑节点

      • 完善项目滚动及双周计划的制定，推进过程中持续更新计划

        • 计划更新频次

    • 项目风险管理机制

      • 有效识别项目过程中的各类风险

        • 项目风险次数
        • 项目风险等级

      • 持续推进风险问题有效解决

        • 风险问题解决周期

    • 项目资源管理机制

      • 建立项目资源计划，明确资源清单

        • 项目所需人数
        • 项目所需人数各层级占比

      • 实时记录资源使用情况，把控资源使用效率

        • 工时登记及时率

    • 项目进度管理机制

      • 通过系统电子化记录跟踪项目进度，监控项目节点任务完成度

        • 项目工作记录完整度
        • 项目阶段性成果及时完成率

      • 有效识别存在进度滞后的项目，分析滞后原因并督促解决

        • 项目完成时间偏离度
        • 项目预算超支率（by全程/分阶段）

    • 项目质量管理机制

      • 建立阶段性成果评审机制，对项目过程质量进行管理

        • 项目阶段性成果首次验收质量达标率

  • 客户管理机制建设

    • 建立及编写客户管理体系制度

      • 制定大客户筛选及服务标准，并根据内外部环境变化及时修订
      • 制定普通客户（即非KA客户）分级标准与升级降级规则，并定时复核修订

    • 建立客户健康度评估体制

      • 成立客户群体风险评估小组，衡量客户行业集中度，有效降低行业系统风险对公司业务的影响
      • 建立客户财务健康度评估制度，定时评估客户财务状况并及时调整相应的信用政策

  • 供应商管理机制建设

    • 建立及编写供应商管理体系制度

      • 明确供应商准入、升级/降级、淘汰规则及标准，提高集团供应商质量

    • 建立供应链健康度评估体制

      • 设置各类重要原材料的集中采购率红线，控制缓解过于集中的采购现状，分散采购风险

        • 采购

          • 供应商集中度
          • 材料采购占比

      • 定时评估各大供应商对公司的依赖程度，奖励供应弹性较高的供应商，警告或淘汰供应弹性较低的供应商，促进供应商共同成长

        • 采购

          • 供应商依赖度

  • 资产管理体系建设

    • 有效建立资产投资及购置计划

      • 完善大型资产购置的可行性论证

        • 财务

          • 设备预计可使用年限
          • 设备购置总预算

        • 生产

          • 设备预计产能

      • 提升固定资产建设的采购质量与计划效率

    • 有效提升资产日常使用及管理效率

      • 明确统一固定资产管理分类标准（房屋、建筑物、生产设备、运输工具、家具用具、电子设备）

      • 对资产及设备管理建立有效的文档追踪（资产设备台账资产卡片、资产目录信息）

        • 生产

          • 台账维护周期
          • 台账覆盖率

        • 财务

          • 资产卡片覆盖率
          • 文档完整程度

      • 明确固定资产的管理、对接验收、盘点工作的责任主体

      • 明确固定资产职责分离的管理原则（实物资产由业务部门负责管理维护；折旧、报废、资产价值增加由财务部门负责

    • 资产处置及报废管理

      • 明确固定资产处置标准与处置流程（年限已满、结构陈旧等）

      • 综合有效评估固定资产处置方式

        • 财务

          • 资产处置收益
          • 资产折旧率
          • 设备折旧率
          • 非正常报废报损资产率

  • 数据标准管理体系建设

    • 完善数据管理的组织与职责

      • 明确集团层面的数据管理组织职能（集团数据治理委员会）
      • 明确数据相关归口监督管理部门（数据中心）
      • 明确各类静态数据及动态数据的主责业务部门（数据owner）

    • 完善数据管理的标准与办法

      • 完善数据治理管控总纲及基本制度
      • 完善数据管理各规范细则（数据质量、数据标准、数据安全）
      • 完善各类业务数据的使用说明及规范管理手册

    • 完善主数据标准体系的建设

      • 明确主数据标准应用业务范围（科目、产品、IPD、客户、供应商、代理商、行业|基础数据标准、指标数据标准）
      • 明确数据标准应用领域（分析数据标准统一、业务数据标准统一、财务数据标准统一等）
      • 明确主数据标准应用组织范围（集团范围标准、产业板块范围标准）
      • 建立各类主数据标准应用方案（维度字段、数据所有者、定义部门、适用范围）
      • 明确主数据分发管理流程（发起部门、分发机制、管理部门）

    • 完善数据平台功能的建设及技术架构的标准统一

      • 通过建立标准的数据管理平台，承载落地数据体系及主数据标准的建设（主数据管理与分发、制度管理、质量管理、安全管理等）
      • 完善标准的数据架构建设（数据湖、大数据平台、关系型数据库等）
      • 完善建立标准的数据模型及分析应用域
      • 建立标准的数据集成技术方法（数据采购，提取，数据仓库技术)

  • QHSE体系建设

    • 质量标准体系建设

      • 应用行业质量认证体系

        • 生产

          • 产品标准产品覆盖率
          • 质量标准工序覆盖率
          • 质量管理体系认证覆盖率
          • 质检人员培训次数/时长

        • 财务

          • 行业质量体系应用支出

      • 建立质量内控标准

        • 生产

          • 质量内控文档覆盖率
          • 质量内控文档修订次数

      • 严格执行质量标准

        • 生产

          • 生产检查不合格次数
          • 报废成本金额
          • 生产停工次数

        • 财务

          • 设计评审费
          • 原材料/半成品/产成品检验费
          • 停工损失费用
          • 报废成本金额

        • 售后

          • 索赔次数
          • 退还次数
          • 保修次数
          • 保修费用
          • 索赔费
          • 退货/换货费用

    • 员工健康管理体系建设

      • 进行定期场地职业卫生监测并进行超标整改工作
      • 建立劳防用品盘点机制并及时补充或更新换代
      • 定期组织员工体检并及时干预治疗异常结果员工

    • 安全管理体系建设

      • 定期场地及设施安全隐患评估（包括消防安全）
      • 制定现场生产工作安全管理制度并严格执行
      • 强化入职安全培训及考核

    • 环境保护体系建设

      • 严格遵守国家及地方政府废弃物排放标准，落实清洁排放责任至各厂
      • 制定施工管理制度，减少施工场地环境污染（扬尘、噪音、生活污水、植被破坏）
      • 完善供应商准入机制，添加环保要求
      • 试点建立清洁生产体系，助力可持续发展

  • 生产标准化体系建设

    • 技术标准

      • 完善标准BOM体系建设，完善BOM的版本管理，明确口径及使用规范
      • 制定测量检验技术标准，提高标准化检验规范程度
      • 制定安装交付技术标准，提高客户现场安装调试质量
      • 统一服务技术标准并培训售后服务技术人员，提高售后维保质量

    • 作业标准

      • 制定生产标准作业指导书，并定期修订

        • 生产

          • 生产作业标准覆盖率
          • 标准修订频率

      • 加强对车间员工标准化作业的培训，减少实际操作与标准操作的差异

    • 其他管理标准

      • 严格遵守国家工业数据质量规范，加强对工业数据采集、传输、维护和使用过程中的质量管理（信息管理标准）
      • 制定职业安全健康管理标准，组织职业危害防治检查并落实职业危害因素整改

  • 组织发展体系建设

    • 保证组织结构的合理性及灵活度

      • 设计高效的组织结构，保持较高的沟通与决策效率（例如扁平化结构）
      • 明确各层级/岗位的分工与职责，提高工作协作效率
      • 建立组织间人才流动机制，提高人岗匹配度

    • 保证组织结构的健康度

      • 通过定期组织宣讲机制，统一员工组织价值观并引导践行
      • 通过建立学习创新小组，培养学习氛围并加强组织创新意识
      • 通过引导及考核手段明确组织中个体的责任及权限，提高员工参与决策的积极性

  • 人才建设体系建设

    • 分析长期内部人才发展需求

      • 有效收集集团各产业人才缺乏信息（职能、级别）
      • 集团统一甄别分析需求信息，确认人才发展需要（具体到岗位）

    • 完善员工发展培养计划

      • 设计不同职能员工的培养方案（管培生轮岗机制等）
      • 跟踪培养方案的执行过程

    • 有效建立人才梯队

      • 透明清晰的晋升机制，留任优秀人才，并保持合理人员结构

        • 人力

          • 人员流失率
          • 人员晋升率

      • 关键岗位人才盘点，及时补缺，确保各核心岗位的人员连续性

    • 建立有效的绩效激励体制

      • 采用多维度绩效考核方式，降低员工短视/次优行为（例如平衡计分卡）

      • 设立特定创新激励项目，对科研/生产等取得重大突破的员工进行奖励

        • 特殊奖励事项次数
        • 奖励金额
